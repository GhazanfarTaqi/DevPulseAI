# DevPulse AI

An AI-powered platform that analyzes real-world software engineering job market data to help developers make smarter career decisions.

---

## What It Does

- Scrapes job postings from multiple platforms automatically
- Extracts and analyzes in-demand skills from the market
- Predicts salary ranges based on your skills, experience, and location
- Analyzes your resume and gives you a Market Fit Score
- Recommends skills to learn and generates personalized career roadmaps
- Matches you with relevant job postings

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js, Tailwind CSS, Recharts |
| Backend | FastAPI, SQLAlchemy |
| Database | PostgreSQL |
| ML | scikit-learn, XGBoost, VADER/FinBERT |
| Scraping | BeautifulSoup, Scrapy |
| Deployment | Vercel, Railway, Docker |
